# Student Performance Dashboard 📊

An Excel dashboard built to analyze and visualize student academic performance.

## Features
- KPI cards (Total Students, Average Score, Top Performers)
- Pivot Tables for data analysis
- Charts: Bar, Pie, and Line graphs
- Slicers for interactive filtering

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Data Visualization

## What I Learned
- How to build interactive dashboards in Excel
- Data analysis using Pivot Tables
- Visualizing data with different chart types
